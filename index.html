Politique de Confidentialité

Dernière mise à jour : 4 août 2025

En résumé : Cette application ne collecte, ne stocke et ne partage aucune donnée personnelle. Elle utilise uniquement votre connexion internet pour récupérer des informations publiques via des API externes.

1. Introduction

Cette politique de confidentialité décrit les pratiques de EuroFastRide concernant la collecte, l'utilisation et la protection de vos informations personnelles.

2. Collecte de données personnelles

Notre application ne collecte AUCUNE donnée personnelle.

Spécifiquement, nous ne collectons pas :

  • Votre nom, email ou informations de contact
  • Votre localisation géographique
  • Vos photos, contacts ou fichiers
  • Vos identifiants ou mots de passe
  • Vos habitudes d'utilisation ou préférences
  • Aucune information sur votre appareil

3. Utilisation d'internet et des API

Notre application utilise votre connexion internet uniquement pour :

  • Récupérer des données publiques via des API externes
  • Afficher ces informations directement dans l'application

Important : Aucune donnée n'est stockée localement sur votre appareil ni envoyée vers nos serveurs. Les informations récupérées sont temporaires et disparaissent quand vous fermez l'application.

4. Stockage de données

Notre application ne possède pas de base de données et ne stocke aucune information :

  • Aucun stockage local sur votre appareil
  • Aucun stockage sur nos serveurs
  • Aucune sauvegarde de données
  • Aucun historique conservé

5. API et services tiers

Notre application peut faire appel à des API publiques pour récupérer des informations. Ces services ont leurs propres politiques :

  • https://queue-times.com/fr - https://queue-times.com/fr/pages/privacy
  • Les requêtes vers ces API ne contiennent aucune information personnelle vous concernant

6. Permissions de l'application

Notre application demande uniquement les permissions nécessaires :

  • Accès Internet : Pour récupérer les données des API externes
  • Accès réseau : Pour vérifier la connectivité

Aucune autre permission n'est demandée.

7. Partage d'informations

Étant donné que nous ne collectons aucune donnée personnelle, il n'y a aucune information à partager avec des tiers.

8. Sécurité

Bien que nous ne stockions aucune donnée, nous nous engageons à maintenir la sécurité de notre application et à utiliser des connexions sécurisées (HTTPS) pour toutes les communications avec les API externes.

9. Applications pour enfants

Notre application est adaptée à tous les âges car elle ne collecte aucune information personnelle. Elle peut être utilisée en toute sécurité par des enfants sous supervision parentale.

10. Modifications de cette politique

Si nous devions un jour modifier notre approche concernant la collecte de données, nous mettrions à jour cette politique avec une nouvelle date et vous en informerions clairement dans l'application.

11. Vos droits

Puisque nous ne collectons aucune donnée personnelle, il n'y a pas de données à :

  • Consulter
  • Modifier
  • Supprimer
  • Transférer
